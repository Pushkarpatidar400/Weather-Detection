# Weather App using API

This Weather App allows users to get the latest weather updates for a specified city using the OpenWeatherMap API.

## Table of Contents

- [Introduction](#introduction)
- [Usage](#usage)
- [Technologies Used](#technologies-used)
- [Files and Structure](#files-and-structure)


## Introduction

This simple Weather App fetches weather information using the OpenWeatherMap API. Users can enter a city name, click on the "WEATHER" button, and get real-time temperature, humidity, and an icon representing the weather condition.

## Usage

1. Open the `index.html` file in a web browser.
2. Enter the desired city name in the input field.
3. Click on the "WEATHER" button to get weather information for the specified city.
4. The app will display the city name, temperature, humidity, and an icon representing the weather condition.

Additionally, the app has a default city (Ratlam) that loads when the page is opened.

## Technologies Used

- **HTML:** Used for structuring the web page.
- **CSS:** Used for styling and layout.
- **JavaScript:** Used for dynamic behavior and interactions on the client side.
- **jQuery:** JavaScript library for simplified DOM manipulation.
- **OpenWeatherMap API:** Used to fetch weather data.
- **GitHub:** Repository hosting and version control.

## Files and Structure

- **index.html:** Main HTML file for the Weather App.
- **weather.js:** JavaScript file containing functions for fetching weather data.
- **weather.css:** CSS file for styling the Weather App.
- **jquery.js:** JavaScript library for DOM manipulation.
